# ziro-boards

### A setout to clone a popular digital realtime collaboration boards platform - Miro, using Nextjs, TypeScript, Convex, Clerk, Liveblocks and TailwindCSS ✨

<br />

<img width="100%" height="100%" src="https://i.ibb.co/270LF4n/ziro-boards-glimpse.gif" alt="ziro-boards-glimpse" />

## Features

- 🤯 Real-time cursors
- 📝 Real-time layer edit
- ⏱️ Real-time database and collaboration
- 🟢 Real-time presence
- 🗑️ Manage organization and users
- 👨‍👨‍👧‍👦 Real-time Collaboration
- 📱 Responsive design
